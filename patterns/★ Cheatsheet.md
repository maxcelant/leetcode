**Bucket Sort**
- Top K most/least frequent elements.

**Quick Select**
- For K closest/greatest elements (in no particular order).

**Binary Search**
- It's sorted or can be sorted.
- Searching for a value.
- Searching for a range of values.

**Two Pointers**
- It's sorted or can be sorted.
- It's a palindrome.
- You need to merge two sorted lists.

**Sliding Window**
- Find a subset/substring/window that matches a condition in a non-contiguous list.
- Find the largest window that fits some criteria.

**Linked List**
- Know how to use dummy nodes and fast-slow pointers.

**Trees**
- Usually involves top-down or bottom-up DFS.
- Top-down means we need some information about the parent to pass down. Like counting depth.
- Bottom-up means we need information about the children. This is the majority of problems.
